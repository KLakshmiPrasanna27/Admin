# Team Task Manager 🚀

A **Team Task Manager** is a web-based application designed to help teams efficiently manage tasks, assign responsibilities, track progress, and improve collaboration. This platform enables admins and team members to organize workflows, monitor deadlines, and ensure smooth project execution.

## 📌 Features

- 🔐 **User Authentication**
  - Secure Login & Signup functionality
  - Role-based access (Admin & Team Members)

- 📋 **Task Management**
  - Create, update, and delete tasks
  - Assign tasks to team members
  - Set priorities and deadlines

- 👥 **Team Collaboration**
  - Manage team members
  - Track assigned tasks
  - Monitor task completion status

- 📊 **Dashboard**
  - View all tasks in one place
  - Task progress tracking
  - Admin overview panel

- 🔔 **Notifications**
  - Task updates and reminders
  - Status tracking

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/team-task-manager.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd team-task-manager
```

### 3️⃣ Install Dependencies

For Frontend:
```bash
cd frontend
npm install
npm start
```

For Backend:
```bash
cd backend
npm install
npm start
```

## 🔧 Environment Variables

Create a `.env` file in backend folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
